## 📧 功能亮点

### 🚀 多用户邮箱订阅

- **结构清晰**: 统计数据、资讯列表、访问链接一目了然
- **HTML格式**: 支持富文本、链接、样式等

### 🔄 智能推送策略
- **当天资讯**: 推送当天发布的所有AI资讯
- **数据统计**: 包含总量、新增、更新时间等关键信息
- **分类标签**: 清晰的分类和来源标识
- **容错处理**: 完善的错误处理和重试机制

## 🛠️ 技术架构

### 邮件发送系统
```
GitHub Actions (定时任务)
    ↓
数据更新检测
    ↓
读取邮箱订阅配置
    ↓
Node.js + Nodemailer
    ↓
QQ邮箱 SMTP 服务
    ↓
用户邮箱接收
```

### 核心组件
1. **邮件发送脚本**: `scripts/send-email-notification.js`
2. **邮件测试工具**: `scripts/test-email-notification.js`
3. **订阅管理API**: `scripts/email-subscription-api.js`
4. **前端管理页面**: `public/email-subscribe.html`
5. **GitHub Actions集成**: `.github/workflows/update-news.yml`

## 🚀 快速上手

### 1. 邮箱订阅管理
访问 [邮箱订阅页面](https://xianyu110.github.io/ai-news-assistant/email-subscribe.html)：

**功能特色**：
- 📊 **实时统计**: 总订阅者、今日新增、活跃订阅数量
- ➕ **添加订阅**: 输入邮箱和姓名，一键添加订阅
- 📋 **订阅管理**: 查看、启用/禁用、删除订阅
- 📤 **数据导出**: JSON/CSV格式导出，便于备份迁移

### 2. 本地测试
```bash
# 安装依赖
npm install nodemailer

# 测试单个邮箱
npm run test-email your-email@example.com

# 测试多个邮箱
npm run test-email email1@example.com email2@example.com

# 手动发送邮件通知
npm run send-email
```

### 3. 订阅管理命令
```bash
# 添加邮箱订阅
npm run email-subscribers add user@example.com

# 删除邮箱订阅
npm run email-subscribers remove user@example.com

# 查看所有订阅者
npm run email-subscribers list

# 清空所有订阅
npm run email-subscribers clear
```

### 4. GitHub Actions配置
项目已内置邮件通知功能，无需额外配置：
- ✅ **自动触发**: 数据更新时自动发送邮件
- ✅ **手动控制**: 支持手动启用/禁用邮件通知
- ✅ **状态反馈**: 详细的发送状态和错误日志
